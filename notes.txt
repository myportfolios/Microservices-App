******** Microservices ***************
** Steps **
1. Install all necessary npm pkgs --> express, mongoose, etc
2. Setup the database and configure it with application
3. have a list of functionalities to be included in the microservice (CRUD) and specify the routes/endpoints

** Books Services *****
Functionalities are:
                    1. create a Books
                    2. delete a Books
                    3. list a Books
                    4. list all books 

** Customers Services *****
Functionalities are:
                    1. create a Customer
                    2. delete a Customer
                    3. list a Customer
                    4. list all Customers
